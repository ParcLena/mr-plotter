# **mr-plotter**
**Mister plotter** (*mr-plotter*) is an **user-friendly** Python tool to create **paper-ready mass-radius diagrams** with your favourite theoretical models. It also includes the ability to **color-code diagrams** based on any published property collected in the [NASA Exoplanet Archive](https://exoplanetarchive.ipac.caltech.edu/).
